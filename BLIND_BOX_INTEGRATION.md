# 盲盒商店集成到津脉文创商城

## 📋 集成概述

已成功将盲盒商店功能集成到津脉文创商城首页，用户在商城购物时可以方便地访问和购买盲盒。

**文件**: [`src/components/marketplace/BlindBoxSection.tsx`](file:///c:/git-repo/src/components/marketplace/BlindBoxSection.tsx)

**功能特性**:

## 🎯 功能特点

### 视觉设计
- **渐变色彩**: 使用紫色到粉色的渐变，营造神秘感和吸引力
- **稀有度系统**: 不同稀有度对应不同颜色
  - 普通：灰色
  - 稀有：蓝色
  - 史诗：紫色
  - 传奇：金色
- **交互动画**: 
  - 卡片悬停效果（缩放、上浮）
  - 闪光扫过效果
  - 平滑的过渡动画

### 用户体验
- **快捷购买**: 点击卡片直接跳转到盲盒商店购买页面
- **信息展示**: 
  - 盲盒名称和描述
  - 价格（积分）
  - 剩余数量
  - 稀有度标识
- **规则说明**: 清晰的盲盒规则说明，帮助用户了解玩法

### 技术实现
- **懒加载**: 图片使用懒加载优化性能
- **响应式设计**: 适配不同屏幕尺寸
  - 手机端：1 列
  - 平板端：2 列
  - 桌面端：4 列
- **状态管理**: 使用盲盒服务获取数据
- **登录检测**: 未登录用户点击会提示登录

## 🔄 用户流程

1. **浏览商城** → 用户访问津脉文创商城首页
2. **看到盲盒专区** → 在 Hero Banner 下方看到醒目的盲盒专区
3. **选择盲盒** → 点击感兴趣的盲盒卡片
4. **跳转购买** → 跳转到 `/blind-box` 页面进行购买和开启
5. **开启盲盒** → 在盲盒商店页面完成购买和开启流程
6. **获得奖励** → 获得的内容自动添加到创作资源库
