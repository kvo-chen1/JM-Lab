# 津币计费功能集成清单

## 已集成的功能

### 1. 津小脉Agent对话 ✅
- **文件**: `src/pages/create/agent/components/ChatPanel.tsx`
- **消耗**: 10津币/轮
- **功能**: 每轮对话前检查余额，余额不足弹出引导

### 2. 图像生成 ✅
- **文件**: `src/pages/create/components/panels/SketchPanel.tsx`
- **消耗**: 50津币/张（标准）
- **功能**: 文生图、图生图生成前检查余额并扣费

### 3. 视频生成 ✅
- **文件**: `src/pages/create/components/panels/SketchPanel.tsx`
- **消耗**: 
  - 5秒视频: 200津币
  - 10秒视频: 400津币
  - 30秒视频: 1000津币
- **功能**: 根据时长计算费用，生成前检查余额

### 4. 文案生成 ✅
- **文件**: `src/pages/create/AIWriterEditor.tsx`
- **消耗**: 20津币/次（初始生成）
- **功能**: 生成文案前检查余额并扣费

### 5. AI写作助手对话 ✅
- **文件**: `src/pages/create/AIWriterEditor.tsx`
- **消耗**: 10津币/轮
- **功能**: AI助手对话优化文案时扣费

### 6. 语音合成（音频生成）✅
- **文件**: `src/components/VoiceOutputButton.tsx`
- **消耗**: 100津币/次
- **功能**: 语音合成前检查余额并扣费

### 7. 导出纹样图片 ✅
- **文件**: `src/pages/create/components/CanvasArea.tsx`
- **消耗**: 50津币/次
- **功能**: 导出图片前检查余额并扣费

## 核心组件

### 1. 津币服务层
- **文件**: `src/services/jinbiService.ts`
- **功能**: 余额查询、消费、发放、记录查询

### 2. 服务端API
- **文件**: `server/routes/jinbi.mjs`
- **功能**: 提供RESTful API接口

### 3. React Hook
- **文件**: `src/hooks/useJinbi.ts`
- **功能**: 前端状态管理和操作封装

### 4. UI组件
- **JinbiBalance**: 余额显示组件
- **JinbiInsufficientModal**: 余额不足弹窗
- **JinbiCostIndicator**: 消费指示器

### 5. 管理页面
- **文件**: `src/pages/JinbiManagement.tsx`
- **功能**: 津币总览、收支记录、充值套餐
