## 🚀 快速开始

### 环境要求

- **Node.js** `>= 18.0.0`
- **PNPM** `>= 9.0.0` (推荐)
- **Git**

### 安装步骤

```bash
# 1. 克隆项目
git clone https://github.com/kvo-123456/jinmai-lab.git
cd jinmai-lab

# 2. 安装依赖
pnpm install

# 3. 启动开发服务器
pnpm dev
```

> 💡 应用将在 `http://localhost:5173` 启动

### 环境变量配置

创建 `.env` 文件：

```env
# 数据库配置
DB_TYPE="local"                          # local | postgresql | neon_api
DATABASE_URL="postgresql://..."
NEON_API_ENDPOINT="https://..."
NEON_API_KEY="your-api-key"

# JWT 配置
JWT_SECRET="your-secret-key"
JWT_EXPIRES_IN="7d"

# CORS 配置
CORS_ALLOW_ORIGIN="*"
```

### 常用命令

```bash
pnpm dev      # 启动开发服务器
pnpm build    # 构建生产版本
pnpm preview  # 预览生产构建
pnpm lint     # 代码检查
pnpm format   # 代码格式化
```
