# 首页推荐位管理功能 - 测试指南

## 快速开始

### 1. 执行数据库迁移

```bash
# 连接到你的 PostgreSQL 数据库
psql -h localhost -U postgres -d <your_database>

# 执行迁移脚本
\i scripts/migration-home-recommendation.sql

# 或者一行命令
psql -h localhost -U postgres -d <your_database> -f scripts/migration-home-recommendation.sql
```

### 2. 启动开发服务器

如果服务器还未启动:

```bash
pnpm dev
```

### 3. 访问管理页面

1. 打开浏览器访问：`http://localhost:5173/admin`
2. 输入管理员密码（默认：`jm2026`）
3. 点击侧边栏的"首页推荐位管理"菜单
4. 或直接访问：`http://localhost:5173/admin?tab=homeRecommendation`
