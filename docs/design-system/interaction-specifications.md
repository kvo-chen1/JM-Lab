## 二、核心交互场景

### 2.1 统计卡片交互

#### 悬停效果
- **触发**：鼠标移入统计卡片
- **动画**：
  - 卡片上浮 2px (200ms ease-out)
  - 阴影从 md 变为 lg
  - 左侧边框加粗至 6px
- **光标**：变为 pointer

#### 点击筛选
- **触发**：点击统计卡片
- **行为**：
  1. 卡片添加选中状态（背景色变为对应颜色的 50 透明度）
  2. 下方表格自动筛选对应状态的数据
  3. 筛选栏显示当前筛选条件标签
  4. URL 参数更新（便于分享和刷新保留状态）

#### 数据更新动画
- **触发**：统计数据变化
- **动画**：
  - 数字使用 CountUp 动画（500ms）
  - 趋势箭头淡入 + 轻微弹跳

### 2.2 数据表格交互

#### 行悬停
- **触发**：鼠标移入表格行
- **效果**：
  - 背景色变为 gray-50
  - 左侧出现 3px 蓝色指示条
  - 操作按钮从透明变为可见

#### 行点击
- **触发**：点击表格行
- **行为**：
  - 打开商单详情侧边栏/抽屉
  - 当前行添加选中状态（蓝色背景）

#### 排序交互
- **触发**：点击表头
- **行为**：
  1. 首次点击：升序排列，显示 ↑ 图标
  2. 再次点击：降序排列，显示 ↓ 图标
  3. 第三次点击：取消排序
- **动画**：表格行重新排列（300ms ease-out）

#### 批量选择
- **触发**：点击行首复选框
- **行为**：
  - 选中行添加背景色
  - 底部显示批量操作栏
  - 显示已选择 X 项

### 2.3 筛选栏交互

#### 搜索框
- **聚焦**：
  - 边框变为蓝色
  - 显示蓝色阴影光晕
  - 占位符文字上移缩小（可选动画标签）

- **输入**：
  - 实时搜索（防抖 300ms）
  - 显示搜索建议下拉框
  - 高亮匹配文字

- **清除**：
  - 输入后显示清除按钮
  - 点击后清空并聚焦

#### 筛选下拉
- **展开**：
  - 下拉菜单从上方滑入（200ms）
  - 背景添加遮罩层

- **选择**：
  - 选中项添加勾选标记
  - 支持多选（标签形式展示）
  - 即时应用筛选（无需确认按钮）

#### 视图切换
- **列表视图**：
  - 详细表格展示
  - 适合数据对比

- **网格视图**：
  - 卡片式布局
  - 适合视觉浏览
  - 切换动画：淡入 + 缩放

### 2.4 空状态交互

#### 首次访问
- **动画序列**：
  1. 插图从下方淡入上移（400ms）
  2. 标题淡入（延迟 100ms）
  3. 描述淡入（延迟 200ms）
  4. 按钮淡入 + 轻微弹跳（延迟 300ms）

#### 引导操作
- **主按钮悬停**：
  - 按钮放大 1.02 倍
  - 阴影加深
  - 图标旋转 15 度

- **点击**：
  - 跳转到发布商单页面
  - 或打开发布弹窗

### 2.5 分页交互

#### 页码切换
- **点击页码**：
  - 当前页高亮显示
  - 表格区域显示加载动画
  - 新数据淡入（300ms）
  - 平滑滚动到表格顶部

#### 每页数量切换
- **展开下拉**：
  - 选项：10/20/50/100
  - 选择后立即刷新数据

## 三、微交互细节

### 3.1 按钮交互

#### 主按钮
```
默认状态 → 悬停 → 点击 → 释放
   ↓         ↓       ↓       ↓
 蓝色背景   深蓝背景  缩放0.98 恢复原状
 无阴影    蓝色阴影   颜色加深
```

#### 图标按钮
- **悬停**：图标颜色加深，背景出现圆形底色
- **点击**：图标缩放 0.9，产生涟漪效果

### 3.2 表单交互

#### 输入框
- **聚焦**：
  - 边框颜色过渡（200ms）
  - 显示蓝色阴影环
  - 标签上浮（如果是浮动标签）

- **错误**：
  - 边框变红
  - 显示错误图标
  - 下方显示错误提示（滑入动画）

- **成功**：
  - 显示绿色勾选图标
  - 边框变绿（可选）

#### 下拉选择
- **展开**：
  - 选项列表从上方展开
  - 选中项高亮
  - 支持键盘导航（↑↓选择，Enter确认）

### 3.3 加载状态

#### 骨架屏
- **动画**：从左到右的渐变扫描效果
- **时长**：1.5s 循环
- **元素**：统计卡片、表格行均有骨架占位

#### 加载更多
- **触发**：滚动到底部
- **显示**：底部加载指示器旋转
- **动画**：新数据从下方滑入

#### 按钮加载
- **状态**：显示旋转图标 + "加载中..."
- **禁用**：防止重复点击

### 3.4 反馈提示

#### Toast 通知
- **位置**：右上角
- **动画**：
  - 进入：从右侧滑入 + 淡入
  - 退出：向上滑出 + 淡出
- **时长**：3秒后自动消失
- **类型**：
  - 成功：绿色，勾选图标
  - 错误：红色，错误图标
  - 警告：黄色，警告图标
  - 信息：蓝色，信息图标

#### 确认对话框
- **触发**：删除等危险操作
- **动画**：
  - 背景遮罩淡入
  - 对话框从中心缩放进入（带弹跳）
- **按钮**：
  - 取消：次要按钮
  - 确认：危险红色按钮

## 四、键盘交互

### 4.1 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| `/` 或 `Ctrl+K` | 聚焦搜索框 |
| `Ctrl+N` | 新建商单 |
| `Esc` | 关闭弹窗/取消操作 |
| `?` | 显示快捷键帮助 |

### 4.2 表格快捷键

| 快捷键 | 功能 |
|--------|------|
| `↑/↓` | 选择上一行/下一行 |
| `Enter` | 打开选中行详情 |
| `Space` | 选中/取消选中行 |
| `Ctrl+A` | 全选当前页 |

### 4.3 焦点管理

- **Tab 顺序**：遵循视觉顺序，从左到右，从上到下
- **焦点样式**：2px 蓝色轮廓线，2px 偏移
- **焦点陷阱**：弹窗内循环，关闭后返回触发元素

## 六、动效时间规范

### 6.1 时长标准

| 类型 | 时长 | 使用场景 |
|------|------|----------|
| Micro | 100-150ms | 按钮点击、颜色变化 |
| Fast | 200ms | 悬停效果、小元素动画 |
| Normal | 300ms | 卡片动画、页面过渡 |
| Slow | 400-500ms | 模态框、复杂动画 |
| Page | 500-700ms | 页面切换、大区域动画 |

### 6.2 缓动函数

```css
/* 标准缓动 */
--ease-standard: cubic-bezier(0.4, 0, 0.2, 1);

/* 减速缓动（进入） */
--ease-decelerate: cubic-bezier(0, 0, 0.2, 1);

/* 加速缓动（退出） */
--ease-accelerate: cubic-bezier(0.4, 0, 1, 1);

/* 弹性缓动 */
--ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);

/* 弹簧缓动 */
--ease-spring: cubic-bezier(0.175, 0.885, 0.32, 1.275);
```

### 6.3 交错动画

列表加载时，每个项目延迟 50ms：
```
项目1: 0ms
项目2: 50ms
项目3: 100ms
项目4: 150ms
...
```

## 八、异常处理交互

### 8.1 网络错误
- **检测**：请求失败
- **显示**：
  - 顶部红色 Banner 提示
  - 表格区域显示错误状态插图
  - 提供"重试"按钮

### 8.2 空搜索结果
- **触发**：搜索无匹配
- **显示**：
  - 提示"未找到相关商单"
  - 建议：检查关键词或清除筛选
  - 提供"清除筛选"快捷按钮

### 8.3 权限不足
- **触发**：尝试无权限操作
- **显示**：
  - Toast 提示权限不足
  - 禁用相关操作按钮
  - 提供升级/联系管理员入口

## 九、性能优化

### 9.1 动画性能
- 使用 `transform` 和 `opacity` 实现动画
- 避免触发布局重排的属性（width, height, top, left）
- 使用 `will-change` 提示浏览器优化
- 动画元素添加 `transform: translateZ(0)` 启用 GPU 加速

### 9.2 减少动效
```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

### 9.3 虚拟滚动
- 数据量 > 100 条时启用
- 只渲染可视区域 + 缓冲区的行
- 滚动时动态更新内容
