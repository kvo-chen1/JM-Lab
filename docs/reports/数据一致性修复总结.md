## 🎯 核心功能

### 时间格式化工具

#### 主要功能

```typescript
import { DateFormatter, DateFormat } from '@/utils/dateFormatter';

// 1. 解析任意格式日期
const date = DateFormatter.parse('2024-01-15T10:30:00.000Z');

// 2. 格式化为指定格式
DateFormatter.format(date, DateFormat.ISO);           // ISO 格式
DateFormatter.format(date, DateFormat.DATE);          // 日期格式
DateFormatter.format(date, DateFormat.TIME);          // 时间格式
DateFormatter.format(date, DateFormat.DATETIME);      // 日期时间
DateFormatter.format(date, DateFormat.CHINESE);       // 中文格式
DateFormatter.format(date, DateFormat.TIMESTAMP);     // 时间戳 (ms)

// 3. 获取当前时间
DateFormatter.now();              // ISO 格式
DateFormatter.nowTimestamp();     // 毫秒时间戳
DateFormatter.nowTimestampSeconds(); // 秒级时间戳

// 4. 相对时间
DateFormatter.relativeTime(date, 'zh');  // "3 小时前"
DateFormatter.relativeTime(date, 'en');  // "3 hours ago"

// 5. 日期操作
DateFormatter.compare(date1, date2);     // 比较
DateFormatter.diff(date1, date2, 'd');   // 差异
DateFormatter.add(date, 7, 'd');         // 加减
DateFormatter.isToday(date);             // 判断

// 6. 日期验证
DateFormatter.isValid(date);             // 是否有效
```

#### 使用场景

- ✅ 数据库时间戳存储
- ✅ 前端时间显示
- ✅ 相对时间展示
- ✅ 时间筛选和排序
- ✅ 时间计算和验证

    persist: true,
