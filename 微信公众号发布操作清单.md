## 二、微信公众号后台操作步骤

### 2.1 登录与新建

1. 登录微信公众号后台（mp.weixin.qq.com）
2. 点击左侧菜单「内容与互动」→「图文消息」
3. 点击「新建图文消息」

### 2.2 编辑文章

#### 步骤1：设置封面和标题

1. **标题**：输入《津脉智坊功能全解析：8大模块助你成为津门文化创作者》
2. **封面图**：上传900×383px封面图
3. **摘要**：输入"从AI创作到商业变现，一站式津门文化共创平台使用指南"

#### 步骤2：编辑正文

按照以下顺序粘贴内容：

1. **开篇引言**
   - 插入用户痛点场景图
   - 粘贴引言文字

2. **平台简介**
   - 插入平台首页截图
   - 粘贴简介文字

3. **核心功能详解**（8个功能模块）
   - 每个功能按以下格式编辑：
     ```
     【功能图标】功能名称
     
     [左侧插入功能截图]
     
     【功能介绍】
     功能说明文字
     
     【核心能力/功能】
     • 能力1
     • 能力2
     • 能力3
     
     【使用场景】
     ✓ 场景1
     ✓ 场景2
     ✓ 场景3
     ```
   - 每个功能之间添加分隔线

4. **谁适合使用津脉智坊**
   - 插入用户类型示意图
   - 粘贴5类用户场景

5. **3步上手津脉智坊**
   - 插入3步流程图
   - 粘贴操作步骤

6. **新用户专属福利**
   - 插入福利海报+二维码
   - 粘贴福利内容

7. **结尾**
   - 粘贴品牌slogan和互动话题
   - 添加联系方式

#### 步骤3：排版设置

1. **字体设置**
   - 正文：14px
   - 标题：16-18px，加粗
   - 小标题：15px，加粗

2. **颜色设置**
   - 标题：#1E40AF（津门蓝）
   - 正文：#374151（正文灰）
   - 重点：#10B981（功能绿）
   - 福利：#F59E0B（强调橙）

3. **间距设置**
   - 段落间距：1.5倍行距
   - 图片与文字间距：10px
   - 功能模块间距：20px

4. **其他设置**
   - 开启「声明原创」（如需要）
   - 设置「阅读原文链接」：www.jinmai-lab.tech
   - 开启「评论功能」

### 2.3 预览与检查

1. 点击「预览」按钮
2. 发送到手机预览
3. 检查以下内容：
   - [ ] 封面图显示正常
   - [ ] 所有图片清晰
   - [ ] 文字排版整齐
   - [ ] 功能介绍完整
   - [ ] 二维码可扫描
   - [ ] 链接可点击

### 2.4 发布设置

1. **发布方式选择**
   - 立即发布：点击「保存并群发」
   - 定时发布：点击「定时群发」，选择时间

2. **推荐发布时间**
   - 最佳：周二至周四 20:00-22:00
   - 备选：周末 10:00-12:00

3. **群发对象**
   - 全部用户
   - 按标签筛选（如需要）

## 五、常见问题处理

### 5.1 图片问题

**问题**：图片上传后模糊
**解决**：确保图片尺寸符合要求，压缩后重新上传

**问题**：图片显示不全
**解决**：检查图片比例，调整为600×400px

### 5.2 排版问题

**问题**：手机端排版错乱
**解决**：使用公众号编辑器预览功能，针对手机端调整

**问题**：文字间距不一致
**解决**：统一使用编辑器格式刷功能

### 5.3 链接问题

**问题**：阅读原文链接无法点击
**解决**：检查链接格式，确保包含http://

**问题**：二维码无法扫描
**解决**：重新生成二维码，测试后再上传
